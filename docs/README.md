# Documentation Directory

此目录存放项目长期维护的架构与部署类非代码文档。

## 文档索引

- **[development.md](development.md)**: 项目开发与组织规范（分层约定、测试及开发环境指令）。
- **[docker.md](docker.md)**: 生产环境 Docker 镜像构建与部署说明（挂载、缓存、环境变量）。
- **[environment.yml](../environment.yml)**: 针对 Conda 用户的可选开发环境定义。

## 编写与维护指南

为保持文档简明高效，请遵循以下约定：

1. **聚焦核心**: 阐明“为什么这样做”比描述“怎么做”更为重要。
2. **高内聚**: 仅涉及特定子目录的说明应维护在对应目录的 `README.md` 中，避免污染全局文档区。
3. **按需扩展**: 仅在引入复杂的跨目录机制、架构变动或需要为协作者提供重大设计背景时，在此新增文档。
