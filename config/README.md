# Browser Search Config

这个目录放项目级配置文件，不放业务实现代码。

## 当前配置文件

### `browser_search.yaml`

浏览器搜索能力的默认配置，包括：

- 浏览器运行参数
- 搜索缓存参数
- 结果过滤参数

### `logging.yaml`

日志输出与归档配置，包括：

- 日志等级
- 终端输出开关
- 终端彩色输出开关
- 本地文件输出路径
- 按天滚动与保留天数
- MCP 工具参数摘要开关

## 优先级

配置优先级从高到低为：

1. 环境变量
2. 本目录下的 YAML 配置
3. 代码默认值

## 适用原则

- 会影响部署环境的参数，优先放在这里并允许环境变量覆盖
- 会改变功能行为的开关，不要散落在代码常量中
- 敏感信息不应提交到这个目录
